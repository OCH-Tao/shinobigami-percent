# shinobigami-percent

シノビガミで判定を振ったときの成功率を簡単に求めることができます！

# 使い方

## 通常時の判定

1. 目標値を入れます
2. 判定についた修正を入れます
3. スペシャル値を入れます
4. ファンブル値を入れます
5. 出力を押します
6. 成功率、スペシャル値、ファンブル値が表示されます!!

## 大権現使用時の判定

1. 目標値を入れます
2. 判定についた修正を入れます
3. スペシャル値を入れます
4. ファンブル値を入れます
5. 大権現を押します
6. 成功率、スペシャル値、ファンブル値が表示されます!!
